# Čeština do hry Balatro

český jazyk do hry balatro bez potřeby modloaderu, patch nainstaluje přímo do hry

## Instalace

1. Stáhnout zip
2. extrahovat
3. spustit patch.exe nebo patch.py
4. zadat lokaci balatra
5. tlačítko patch
